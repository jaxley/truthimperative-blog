# The Truth Imperative blog

- This blog is built using Hugo
- This blog is using a fork of the [lwojcik/eleventy-template-bliss: Single-column blog template for Eleventy focused on simplicity without sacrificing functionality](https://github.com/lwojcik/eleventy-template-bliss/)
    - Because the theme is no longer maintained, I created a variant of it that works with Hugo and uses almost no javascript. See themes/hugo-bliss. 
- The blog content is copyrighted by me.
- I use Obsidian templates to compose blog posts and manage drafts.
